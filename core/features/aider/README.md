# Aider Feature

> **Coming soon** — This feature is a placeholder for future Aider support.

This devcontainer feature will install and configure [Aider](https://github.com/paul-gauthier/aider), an AI pair programming tool.

## Usage

In your `isolde.yaml`:

```yaml
agent:
  name: aider
  version: latest
  options: {}
```

Then run `isolde sync` to generate the devcontainer configuration.
