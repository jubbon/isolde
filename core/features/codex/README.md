# Codex CLI Feature

> **Coming soon** — This feature is a placeholder for future Codex CLI support.

This devcontainer feature will install and configure the [OpenAI Codex CLI](https://github.com/openai/codex) coding agent.

## Usage

In your `isolde.yaml`:

```yaml
agent:
  name: codex
  version: latest
  options: {}
```

Then run `isolde sync` to generate the devcontainer configuration.
