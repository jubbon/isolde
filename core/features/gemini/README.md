# Gemini CLI Feature

> **Coming soon** — This feature is a placeholder for future Gemini CLI support.

This devcontainer feature will install and configure the [Google Gemini CLI](https://github.com/google-gemini/gemini-cli) coding agent.

## Usage

In your `isolde.yaml`:

```yaml
agent:
  name: gemini
  version: latest
  options: {}
```

Then run `isolde sync` to generate the devcontainer configuration.
